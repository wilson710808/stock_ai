# Stock AI 版本記錄

> 約定：本地與 GitHub 每次提交（含整合 / 修復 / 小功能）都升一個 **patch** 號（v2.0.0 → v2.0.1 → v2.0.2 …）。  
> 重大功能 / 不相容變更才升 minor (v2.x.0)；架構級重寫升 major (vX.0.0)。

版本顯示位置：
- 首頁頂部 header 的 logo 右側（綠色徽章）
- 登入 / 註冊頁標題上方
- 後端 API `GET /api/version` 動態回傳，前端啟動時自動同步
